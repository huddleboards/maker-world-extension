An Unofficial MakerWorld Extension

Support & Feedback

If you would like to see new features or support this project, please download my models or buy me a coffee.

Instructions

On any design page, simply view the download reward points, the approximate gift card value, and if it's a part of contest.

On any user page, you can also view the reward points, gift card value, and if it's part of a contest. For designs you've already liked, it will also show you the hot score.

As with any unofficial extension, we won't know if the MakerWorld developers are deploying changes that could cause this extension to not work. We will try to keep it updated.
